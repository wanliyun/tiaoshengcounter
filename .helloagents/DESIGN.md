# 跳绳计数器 — 设计系统

## 设计简报
- 界面目的：让用户在跳绳过程中一眼看清当前次数
- 目标用户与场景：运动中，手机放在 1.5~2 米前方
- 主要视口：竖屏手机 360~414dp 宽
- 情绪方向：运动科技感 + 极简克制
- 记忆点：巨大的中央数字 + 实时 BPM 指示环

## 颜色系统
| 角色 | 色值 | 用途 |
|------|------|------|
| 背景 | Camera 预览作底 | 全屏摄像头实时画面 |
| 覆盖层表面 | rgba(0,0,0,0.45) | 半透明卡片/HUD 条 |
| 正文 | #FFFFFF | 主要文字 |
| 弱化 | rgba(255,255,255,0.65) | 次要文字 |
| 强调-运动橙 | #FF6B35 | 核心交互、BPM、选中态 |
| 强调-运动绿 | #00D68F | 计数脉冲、成功态 |
| 暂停黄 | #FFC107 | 暂停状态指示 |
| 异常红 | #FF4444 | 错误、结束按钮 |

## 排版
| 角色 | 字号 | 字重 | 字体 |
|------|------|------|------|
| 计数数字 | 72sp | Bold | 等宽数字 |
| 统计数字 | 28sp | SemiBold | 系统默认 |
| 正文标签 | 14sp | Normal | 系统默认 |
| 辅助说明 | 12sp | Normal | 系统默认 |

## 间距与圆角
- 基础间距单位：4dp (4/8/12/16/24/32/48)
- 卡片圆角：16dp
- 按钮圆角：999dp (胶囊)
- 面板圆角：12dp

## 组件状态覆盖
| 状态 | 视觉处理 |
|------|----------|
| IDLE | 引导文字 + 人体轮廓线 |
| READY | 绿色检测框 + "准备开始" |
| COUNTING | 完整 HUD + BPM 环脉冲 + 数字弹跳 |
| PAUSED | 半透明遮罩 + 冻结数字 + 继续/结束按钮 |
| LOW_LIGHT | 顶部黄色警告条 |
| NO_PERSON | 红色闪烁提示 |

## 动效
- 计数弹跳：scale 1.0→1.15→1.0, spring, 200ms
- BPM 环脉冲：scale 1.0→1.05→1.0, 150ms
- 状态过渡：淡入 300ms / 遮罩 400ms
- 遵循 prefers-reduced-motion

## 无障碍
- 按钮最小触摸目标 48dp
- contentDescription 全覆盖
- 支持系统字体缩放 (sp)
- TTS 语音播报作为视觉替代

## 约束
- 字体 ≤2 种（系统默认 + 等宽数字）
- 强调色 ≤2 个（运动橙 + 运动绿）
- 不使用 emoji 作为图标
- 无装饰性动画
