# 跳绳计数器 — 需求文档

## 项目概述
完全离线 AI 跳绳计数器 Android App。用户将手机放置在前方，打开摄像头即可自动识别跳绳动作并实时计数，无需额外硬件。

## 核心需求

### 实时计数
- 双脚跳模式：取左右脚踝 Y 坐标均值进行峰谷检测
- 交替跳模式：左右脚踝独立追踪，任一满足判定即计数
- 两次有效跳跃间隔 ≥150ms（防抖去毛刺）
- 双脚跳准确率 ≥95%，交替跳 ≥90%

### 运动数据
- 自动计时（首次跳跃开始，连续 5 秒无跳跃暂停）
- 实时 BPM 显示（最近 5 秒滑动窗口）
- 卡路里估算（MET 公式，体重可配置）
- 本地历史记录（Room 数据库，按日/周/月聚合）

### 完全离线
- 模型内嵌 APK assets，无网络依赖
- 不声明 INTERNET 权限
- 数据完全不出设备

### 设置与辅助
- TTS 语音播报（可配置间隔/开关）
- 计数灵敏度三档调节
- 前后摄像头切换
- 中英文双语言

## 非功能需求
- 最低 API 26 (Android 8.0)
- APK ≤30MB（含模型 ~5MB）
- 中端机推理 ≥25 FPS，低端机 ≥15 FPS
- 运行时内存 ≤150MB
- 冷启动 ≤2s，首次模型加载 ≤5s

## 成功指标
- [ ] 双脚跳准确率 ≥95%
- [ ] 交替跳准确率 ≥90%
- [ ] 中端机推理 ≥25 FPS
- [ ] 冷启动到计数 ≤3s
- [ ] APK ≤30MB
